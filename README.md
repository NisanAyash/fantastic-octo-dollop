### Let's run the dashboard project
	
	1. cd dashboard
	2. docker-compose up -d
	3. Go to http://localhost:3005/

* You can customize the session expiration through the src/.env file

### chat-app
	
	1. cd chat-app
	2. npm i
	3. node app.js
	4. open a & b html files with Open live server extension
